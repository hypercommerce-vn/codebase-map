CM-S2-06 2 SP

Edge Resolution Improvement

Improve Python parser to resolve self.repo.method() call chains through __init__ type hints. Increases edge accuracy from ~94% to ~99%.

Before / After Comparison
Before (v1.1)
# Source code
class CustomerService:
  def __init__(self, repo: CustomerRepository):
    self.repo = repo

  def create(self, data):
    self.repo.save(data)
Resolved edge:
self.repo.save → unknown.save ✗ unresolved
After (v1.2)
# Same source code
class CustomerService:
  def __init__(self, repo: CustomerRepository):
    self.repo = repo

  def create(self, data):
    self.repo.save(data)
Resolved edge:
self.repo.save → CustomerRepository.save ✓ resolved
Resolution Chain Visualization
self.repo
→
__init__(repo: CustomerRepository)
→
CustomerRepository.save
Parser Accuracy
94% → 99%
Unresolved Edges
497 → 83
Implementation Notes
  • Parse __init__ type annotations to build self.attr -> Type mapping
  • Resolve self.attr.method() chains: look up attr type, find method on that class
  • Also handle: constructor injection, class-level type hints, typing.Optional
  • Modify python_parser.py: add _resolve_attribute_chain() method
  • Store resolution metadata on edges: edge.resolved = True/False
